# General Questions — 50 Rock-Solid Questions Reference

> **Source:** Curated from V1 Core Module (130Q), V2 Introduction Core (100Q), and Master Library (389Q)
> **Purpose:** Populate the Main Module "General Questions" section (currently empty in codebase)
> **Response Type Taxonomy:** See `CLUES_MAIN_BUILD_REFERENCE.md` Section 8
> **Scope:** Questions about the PERSON — personality, readiness, social identity, lifestyle philosophy, and vision. NOT Demographics, DNW, Must Haves, Trade-offs, or specialized module topics.
> **Why these matter:** These help the LLM understand WHO the user is, not just WHAT they want. Two users with identical Demographics/DNW/MH can get completely different recommendations based on these answers.
